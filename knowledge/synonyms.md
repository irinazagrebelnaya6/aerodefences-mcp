# Синоніми EN↔UA для RAG (конфіг, НЕ джерело для цитування)

Цей файл НЕ індексується як звичайний knowledge-документ. Замість цього
`rag_index.py` читає правила нижче й **вписує українські синоніми у документи
товарів з БД** під час індексації — щоб англомовні описи знаходилися за
українськими запитами.

Формат правила (один рядок): `тригери, через кому => українські синоніми`.
Тригер спрацьовує, якщо він є в назві категорії/тексті товару (за словом або
підрядком). Тоді праву частину додають у пошуковий текст товару (невидимо для
користувача, лише для матчингу).
